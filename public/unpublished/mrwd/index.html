Morphological Representation of Wavelet Data:
Review of a Paper by Servetto, Ramchandran and Orchard
by Zvika Ben-Haim

This web page contains simulation results which are part of a report written by Zvika Ben-Haim. The report covered the following paper:

S.D. Servetto, K. Ramchandran, and M.T. Orchard,
``Image coding based on a morphological representation of wavelet data,''
IEEE Trans. Image Proc. 8(9): 1161--1174, Sep. 1999.

The review was written by Zvika Ben-Haim for the course Combined Time-Frequency Representations, in Winter 2003-2004.
